Politique de Confidentialité

Dernière mise à jour : 25 mai 2025

Chez Clément Nuttens, développeur indépendant, nous nous engageons à protéger votre vie privée. Cette politique de confidentialité explique comment vos données sont collectées, utilisées et protégées lorsque vous interagissez avec notre application GPT personnalisée.

1. Données collectées

Lorsque vous interagissez avec notre GPT, les types de données suivants peuvent être collectés :

  • Contenu que vous saisissez (textes, questions, requêtes),
  • Informations techniques transmises automatiquement (adresse IP, navigateur, etc.),
  • Réponses ou journaux techniques générés par notre API (ex : entrecode.site).

2. Utilisation des données

Nous utilisons vos données uniquement pour :

  • Fournir une réponse à votre requête,
  • Améliorer la qualité et les performances de notre service,
  • Assurer la sécurité et la fiabilité de l’infrastructure.

3. Partage des données

Nous ne partageons pas vos données avec des tiers, sauf dans les cas suivants :

  • Prestataires techniques (comme OpenAI ou notre hébergeur),
  • Obligations légales ou judiciaires.

4. Conservation des données

Les données peuvent être conservées temporairement à des fins d’analyse ou de journalisation. Aucune donnée personnelle identifiable n’est stockée durablement par notre service.

5. Vos droits

Conformément au RGPD (si vous êtes résident de l'Union Européenne), vous avez le droit de :

  • Demander l'accès à vos données,
  • Demander leur modification ou leur suppression,
  • Contacter notre équipe pour toute demande liée à la vie privée.
